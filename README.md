# استبيان ADHD - ADHD Survey

استبيان تفاعلي لتقييم أعراض اضطراب فرط الحركة وتشتت الانتباه (ADHD)

## 🧠 نبذة عن المشروع

هذا المشروع عبارة عن استبيان تفاعلي يحتوي على 25 عرض من أعراض ADHD مقسمين على 3 فئات:

1. **أعراض تشتت الانتباه** (9 أعراض)
2. **أعراض الاندفاعية وفرط الحركة** (9 أعراض)
3. **أعراض غير رسمية/إضافية** (7 أعراض)

## 📊 نظام النقاط

### التكرار (Frequency)
| الخيار | النقاط |
|--------|--------|
| نادرًا | 1 نقطة |
| عدة مرات في الشهر | 2 نقطة |
| عدة مرات في الأسبوع | 3 نقاط |
| عدة مرات في اليوم | 4 نقاط |

### مجالات التأثير (Impact Areas)
- كل مجال محدد = **1 نقطة** (حد أقصى 3 نقاط)
  - العمل - الدراسة
  - العائلة - المنزل
  - الأصدقاء - الهوايات

### المدة (Duration)
| الخيار | النقاط |
|--------|--------|
| لأقل من ستة أشهر | 1 نقطة |
| لأكثر من ستة أشهر | 2 نقطة |
| طوال حياتي | 3 نقاط |

**الحد الأقصى لكل عرض:** 10 نقاط (4 + 3 + 3)

## 🛠️ التقنيات المستخدمة

- **Framework:** Next.js 16 مع App Router
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS 4
- **UI Components:** shadcn/ui
- **Database:** Prisma ORM مع SQLite
- **State Management:** Zustand

## 📦 التثبيت والتشغيل

### المتطلبات
- Node.js 18+ أو Bun 1.0+
- SQLite (مدمج)

### خطوات التثبيت

```bash

unzip file.zip

# 2. الدخول للمجلد

# 4. إعداد قاعدة البيانات
bun run db:push
# أو
npm run db:push

# 5. تشغيل المشروع
bun run dev
# أو

```

افتح المتصفح على: `http://localhost:3000`
